# Bug Reforting Guidelines

PostgreSQL의 버그를 만약 찾게된다면 우리는 그것에 대해 전달받길 원한다. 버그 리포팅은 PostgreSQL을 더 안정적이게 만들 수 있는 중요한 부분이다. 왜냐면 아무리 주의룰 기울여도 PostgreSQL의 모든 부분이 모든 상황의 모든 플랫폼에서 정상 작동한다고 보장할 순 없다.
