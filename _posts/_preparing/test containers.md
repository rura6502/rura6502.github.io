# Test Containers
애플리케이션의 기능을 테스트할 때 데이터베이스나 다른 여러 컴포넌트와 엮인 기능의 경우 JUnit등의 테스트 프레임워크를 사용해서 테스트를 하기가 곤란한 경우가 많다.
