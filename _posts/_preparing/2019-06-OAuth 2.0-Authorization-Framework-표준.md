OAuth 2.0 authorization framework는 HTTP 요청에 대한 엑세스 제한을 구현하게 해줌. 크게 두가지의 기능을 제공함.

1. Resource Owner를 대신해서 들어온 HTTP Service 요청을 허용할지 말지에 대하여 결정
2. Third-part application이 엑세스 권한을 스스로 얻을수 있도록(이후 기술할 토큰 방식)
