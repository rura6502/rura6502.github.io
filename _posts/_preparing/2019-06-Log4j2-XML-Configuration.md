Log4j2의 XML 설정 파일은 아래와 같은 구조로 구성되어 있음.

```xml
<Configuration>
	<Appenders>
		<!-- define Appender --!>
	</Appenders>
	<Loggers>
		<!-- define Logger --!>
	</Loggers>
</Configuration>
```
